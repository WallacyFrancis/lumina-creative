## ğŸ“– Project Overview

**Lumina Creative** is a high-end corporate website designed for a digital agency. Unlike static templates, this project is a fully functional web application connected to a database.

It demonstrates how to build a **CMS-like architecture** using Next.js and Supabase, allowing for dynamic portfolio management and real-time lead capture via contact forms using Server Actions.

### âœ¨ Key Features

* **Dynamic Portfolio:** Project data (images, titles, tags) is fetched server-side from a **Supabase (PostgreSQL)** database.
* **Server Actions:** Contact form submissions are handled directly on the server, saving leads to the database without API routes.
* **Modern UI/UX:** "Awwwards-style" aesthetic with elegant typography (Playfair Display), whitespace usage, and responsive design.
* **Mobile-First:** Fully animated mobile navigation drawer.
* **Performance:** Optimized images, strict TypeScript typing, and Next.js App Router architecture.

## ğŸš€ Getting Started

### Prerequisites

* Node.js 18+
* A Supabase account

### Installation

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/YOUR-USERNAME/lumina-creative.git](https://github.com/YOUR-USERNAME/lumina-creative.git)
    cd lumina-creative
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Configure Environment Variables:**
    Create a `.env.local` file in the root directory:
    ```bash
    NEXT_PUBLIC_SUPABASE_URL=your_supabase_project_url
    NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key
    ```

4.  **Database Setup (SQL):**
    Run the following query in your Supabase SQL Editor to create the necessary tables:

    ```sql
    -- Projects Table
    create table projects (
      id bigint generated by default as identity primary key,
      created_at timestamp with time zone default timezone('utc'::text, now()) not null,
      title text not null,
      description text not null,
      image_url text,
      tags text[],
      client_name text
    );

    -- Contacts Table
    create table contacts (
      id bigint generated by default as identity primary key,
      created_at timestamp with time zone default timezone('utc'::text, now()) not null,
      name text not null,
      email text not null,
      message text not null
    );
    ```

5.  **Run the development server:**
    ```bash
    npm run dev
    ```
